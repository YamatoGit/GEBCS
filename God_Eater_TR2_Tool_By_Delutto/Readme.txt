1 - Usage

 1.1 - Parameters: [option] <File.tr2> <OutputFolder>
				   Options: -e = Export Files
                            -i = Import Files
                            -h = This Help
		 
 1.2 - Export: -e
       God_Eater_TR2_Tool.exe -e 00000004_00025A84A5B46946.tr2 00000004_00025A84A5B46946
 
 1.3 - Import: -i (Point to original file)
       God_Eater_TR2_Tool.exe -i 00000004_00025A84A5B46946.tr2 00000004_00025A84A5B46946
	   Will be created a "00000004_00025A84A5B46946.tr2.NEW" file.
	   
 1.4 - Or just run the tool from executable, a dialog option will be showed.

 
2 - Notes

 2.1 - Both options use the same parameters order, because the tool need some unknown infos from original
       TR2 file to create a new one.
 
 2.2 - Keep only the mod files on extracted folder, there's no need to import all extracted files.
